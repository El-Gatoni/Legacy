Informations
• Installer l’application
Android : ouvrez Chrome, appuyez sur le menu ⋮ puis Ajouter à l’écran d’accueil.
iPhone / iPad : ouvrez Safari, appuyez sur le bouton Partager puis Sur l’écran d’accueil.
• Suivi de patrimoine privé
Legacy est une application de suivi de patrimoine personnel. Toutes vos données sont stockées uniquement dans le cache de votre navigateur, sur votre appareil. Aucun compte, aucun serveur, aucune publicité.
• Alertes d’échéance
Legacy affiche un bandeau discret sur l’écran d’accueil lorsqu’un prêt ou un crédit immobilier arrive à échéance dans moins d’un mois ou moins de six mois. Cliquez sur le bandeau pour accéder directement à l’onglet Passif.
• Notifications push
Les notifications push sont absentes de l’application. Legacy protège vos données en les conservant exclusivement sur votre appareil — aucun compte, aucun serveur, aucune transmission externe. Les notifications push en arrière-plan nécessitent un serveur externe qui connaît vos données pour savoir quand vous alerter. Ce fonctionnement est incompatible avec la philosophie de Legacy.
• Code PIN
Le code PIN s’affiche à chaque ouverture de l’application. Il est conçu pour protéger le contenu de Legacy des regards involontaires — par exemple si quelqu’un clique sur l’icône par mégarde. Disponible en 4 ou 6 chiffres. Il ne chiffre pas vos données et ne remplace pas une protection système.

⚠️ En cas d’oubli, la seule option est de réinitialiser complètement l’application depuis l’écran PIN. Toutes vos données seront effacées. Exportez régulièrement votre sauvegarde JSON depuis Paramètres pour pouvoir les restaurer. Activez-le dans Paramètres → Configuration → Code PIN.
• Graphique d’évolution
Un instantané de votre patrimoine net et brut est enregistré une fois par mois, automatiquement, à chaque ouverture de l’application. L’historique est conservé sur 24 mois. La courbe se construit progressivement — les données apparaissent dès le deuxième mois d’utilisation.
• Sauvegarde JSON
Le cache navigateur peut être effacé lors d’une mise à jour ou d’une réinitialisation de l’appareil. Exportez régulièrement vos données en JSON depuis Paramètres pour ne rien perdre. Ce fichier permet de restaurer l’application à l’identique sur n’importe quel appareil.
• Modes de calcul
Patrimoine net = actifs bruts − dettes en cours.
PRU (Prix de Revient Unitaire) = total investi ÷ nombre de parts. Peut inclure ou exclure les frais selon le réglage dans Configuration.
Plus-value latente = (cours actuel − PRU) × nombre de parts.
Gain net immobilier = (estimation × part) − (crédit × part) + solde compte − (frais d’acquisition × part) − approvisionnements. Les frais d’acquisition (apport, notaire, travaux...) sont pondrérés par votre quote-part de détention.
Allocation = répartition en pourcentage des actifs investis par classe d’actif.
• Onglet Immobilier
Chaque bien est défini par sa surface, son prix au m² et sa quote-part (votre pourcentage de détention). Tous les calculs — estimation, crédit restant, gain à la vente — sont automatiquement pondérés par ce pourcentage.

Les unités locatives permettent de détailler chaque espace loué (appartement, dépendance, local…) avec son loyer et ses charges propres. Le rendement locatif net est calculé automatiquement.

Le crédit immobilier et les frais d’acquisition se saisissent dans l’onglet Passif, sur la fiche du bien. Les frais d’acquisition (apport, notaire, travaux...) sont pris en compte dans le calcul du gain net à la vente.
• Onglet Bourse
Les cours sont récupérés automatiquement via Yahoo Finance en appuyant sur « Actualiser les cours ». Pour les titres cotés sur Euronext Paris, saisissez uniquement le nom du symbole sans suffixe — l’application ajoute .PA automatiquement (ex : LVMH, BNP, AIR).

Le pourcentage affiché sur chaque carte titre représente le poids de ce titre dans votre portefeuille total — c’est-à-dire sa valeur actuelle divisée par la valeur totale de tous vos titres.
• Onglet Bitcoin
Enregistrez chaque achat de bitcoin comme une transaction : le montant investi et la quantité de BTC acquise s’accumulent dans une pool totale. Vous pouvez ensuite allouer ce bitcoin à différents wallets (hardware, exchange, self-custody…) pour suivre la distribution de vos BTC.
